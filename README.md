## Requirements

- GCC/Clang (or MSVC)

## Build

### Linux (GCC)
```bash
gcc -Wall -Wextra -Wpedantic -std=c11 *.c -o wordle
```

### macOS (Clang / GCC alias)
```bash
gcc -Wall -Wextra -Wpedantic -std=c11 *.c -o wordle
```

### Windows (MinGW / MSYS2)
```bash
gcc -Wall -Wextra -Wpedantic -std=c11 *.c -o wordle.exe
```

### Windows (MSVC / cl)
```bash
cl /W4 /TC *.c
```

---

## Run

### Linux / macOS / WSL / MSYS2
```bash
./wordle
```

### Windows (MinGW / MSYS2)
```bash
wordle.exe
```

### Windows (MSVC)
```bash
wordle.exe
```
